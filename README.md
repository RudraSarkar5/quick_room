# quick_room

QuickRoom is a simple data-sharing application that allows users to share data quickly without creating an account or providing any credentials. Users can create a temporary room, upload media files or documents up to 50 MB, and share the room credentials with others. Anyone with the room details can access the room and download the shared data instantly. The goal of QuickRoom is to provide a fast, simple, and hassle-free way to exchange data between users.
